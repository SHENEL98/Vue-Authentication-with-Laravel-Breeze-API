## Backend - Laravel Breeze API

- Install breeze in laravel
- Install breeze api 
- Change .env
- FRONTEND_URL=http://localhost:3000
- SANCTUM_STATEFUL_DOMAINS=localhost:3000
- SESSION_DOMAIN=localhost
